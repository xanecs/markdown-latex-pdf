# Markdown-Latex-PDF

This is an example of my Markdown-Latex-PDF workflow using pandoc

Requirements:
* LaTeX (e.g TeXlive, MacTeX or MikTeX)
* Pandoc (ask your local package dealer)

Usage:
0. Adjust config.tex
1. Write markdown in src directory
2. Run `./build.sh`
3. Look at latex/main.pdf
